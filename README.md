# Fallout 76 Item Finder Map v76.READY

Live → https://0mrcrazy0.github.io/fallout76-itemfindermap/

simple, free, no-login map to mark and track anything in Fallout 76:  
magazines, bobbleheads, plans, treasure maps, junk, power armor… whatever you want.

### What it does
- double click the map to log markers
- unlock drag to move, lock when happy
- filter by category
- save everything in your browser
- export / import your markers (JSON)
- download community discoveries others share
- works offline, dark mode & mobile-friendly
- click the How to Use button in the tools section for more info

just open the link and start logging locations. No account needed.
