# Fallout 76 Item Finder Map v76.READY

**Live version:** https://0mrcrazy0.github.io/fallout76-itemfindermap/

A simple, free, no-login browser-based map tool for Fallout 76 players to mark and track anything in Appalachia — magazines, bobbleheads, plans, treasure maps, junk nodes, power armor spawns, or any custom locations you discover.

### Features

- **Double-click** (or double-tap on mobile) anywhere on the map to log a new permanent marker
- **Unlock** markers to drag and reposition them, then lock when satisfied
- Filter markers by category (default + your own custom ones)
- All data saved locally in your browser — no account required
- **Export / Import** your markers as JSON (share with friends or backup easily)
- **Download Community Map** — periodically update to include markers submitted by other players
- Earn **XP** and level up for every marker you log
- Fully **offline-capable** after first load
- Mobile-friendly with touch gestures (long-press, single-tap, double-tap)

### How to Use

1. Open the live link: https://0mrcrazy0.github.io/fallout76-itemfindermap/
2. Click **"How to Use"** in the tools panel for full instructions (includes PC vs mobile controls)
3. Start double-clicking locations to log markers
4. Use the **Search Bar** to filter by description, category, or grid (e.g. "C4")
5. Click **"Update Community Map"** occasionally to get the latest community discoveries

Enjoy exploring Appalachia — and happy logging!
