Fallout 76 Item Finder Map Info

PC: Double-left-click the map to log a new permanent marker.
Mobile: Double-tap the map to log a new permanent marker.

PC: Right-click the map to open the Postcard menu → type a message → click Copy Link to share a temporary postcard (expires in 5 minutes).
Mobile: Long-press the map to open the Postcard menu → type a message → tap Copy Link.

New markers are automatically LOCKED and glow green for 2 minutes.

Unlock a marker in its popup or edit modal to allow dragging.

Drag any unlocked marker — drop to auto-save the new position (it will glow and re-lock for safety).

PC: Left-click any marker to open its popup (community) or Edit menu (yours/kept).
Mobile: Single-tap any marker to open its popup or Edit menu.

PC: Left-click a row in the Content Inventory to instantly zoom to that marker location.
Mobile: Single-tap a row in the Content Inventory to instantly zoom to that marker location.

PC: Right-click a row in the Content Inventory:
Mobile: Long-press a row in the Content Inventory:

  • Your markers (created or kept) → opens the Edit modal + zooms to the location
  • Community markers → opens the marker popup + zooms to the location

Use the Search Bar to filter by description, category, or grid coordinates (e.g. C4).

The Toggle Map button switches between the named map and the clean (no-name) map.

Click/tap anywhere on the map → displays the current grid coordinate in a green popup.

The Grid button toggles the 10×10 grid overlay on or off.

Create and manage Custom Categories in the Toggle Categories menu.

Export / Import your markers — duplicates are automatically skipped.

Save Map As JPEG captures your current view with all visible markers.

Earn XP for every marker you place — level up like in the game!

Lose XP when deleting markers.

The map works fully offline after the first load (ideal for areas with no signal).

Periodically tap Download Community Map to receive the latest community updates.
